# Éclairage

## nhecl.csv

Nombre moyen d'heure d'éclairage par an et par zone climatique.

**Schéma :**

|      Champs       |               Description                |              Type              |
| :---------------: | :--------------------------------------: | :----------------------------: |
| `zone_climatique` |             Zone climatique              | `enum(climat:zone-climatique)` |
|      `nhecl`      | Nombre moyen d'heure d'éclairage par (h) |            `number`            |

**Références réglementaires :**

- Méthode 3CL-DPE - §16.1
